# Pinned dependencies for the architecture-sweep sidecar venv (`.venv-mlx`).
#
# These are the exact versions installed when this PR's results.csv was
# produced — pinning them means a contributor re-running the sweep months
# later won't hit a different `jax-mlx-plugin` or `jax` version with
# different lowering coverage and a different basin landscape.
#
# Usage:
#
#     uv venv --python 3.13 .venv-mlx
#     .venv-mlx/bin/python -m ensurepip --upgrade
#     .venv-mlx/bin/python -m pip install --timeout 600 -r scripts/requirements-mlx.txt
#     .venv-mlx/bin/python -m pip install --timeout 600 -e .
#     .venv-mlx/bin/python -c "import jax; print(jax.devices())"   # [mlx:0]
#
# The trailing editable-ondes install is intentionally separate from this
# file so users can install against any branch of `ondes` without editing
# the pin set. `mlx-metal` is the ~38 MB wheel that historically times out
# under `uv pip install`'s default 30-second timeout — pip's
# `--timeout 600` covers it; uv's `UV_HTTP_TIMEOUT=600` does not always
# (see writeup Anomalies).
